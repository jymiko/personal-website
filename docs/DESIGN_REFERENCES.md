## Trend Teknis 2025/2026 — Yang Benar-Benar Baru

### 1. CSS Scroll-Driven Animations (Native Browser — TANPA JS)
Ini trend terbesar 2025. Chrome 115+ dan browser modern sudah support native.
Tidak perlu GSAP atau Framer Motion untuk scroll animation dasar.

```css
/* Elemen fade-in saat masuk viewport — pure CSS */
@keyframes fadeUp {
  from { opacity: 0; translate: 0 40px; }
  to   { opacity: 1; translate: 0 0; }
}

.reveal {
  animation: fadeUp linear both;
  animation-timeline: view();
  animation-range: entry 0% entry 30%;
}

/* Progress bar scroll — pure CSS */
.scroll-progress {
  position: fixed;
  top: 0; left: 0;
  height: 3px;
  background: var(--accent-blue);
  transform-origin: left;
  animation: scaleX linear;
  animation-timeline: scroll(root block);
}
@keyframes scaleX {
  from { transform: scaleX(0); }
  to   { transform: scaleX(1); }
}
```

### 2. View Transitions API (Page Transitions Buttery Smooth)
```css
/* Aktifkan view transitions */
@view-transition { navigation: auto; }

/* Custom transition per elemen */
.hero-title {
  view-transition-name: hero-title;
}

@keyframes slide-in {
  from { translate: 0 -20px; opacity: 0; }
}
::view-transition-new(hero-title) {
  animation: slide-in 0.4s ease;
}
```

### 3. OKLCH Color Space (Modern CSS Color)
Lebih perceptually uniform, bisa generate palette programatik.
```css
:root {
  /* OKLCH: lightness chroma hue */
  --accent-blue:   oklch(65% 0.2 250);   /* biru */
  --accent-purple: oklch(65% 0.18 300);  /* purple */
  --accent-green:  oklch(72% 0.17 160);  /* green */

  /* Generate shade otomatis dengan CSS relative color */
  --accent-blue-dim: oklch(from var(--accent-blue) calc(l - 0.2) c h);
  --accent-blue-glow: oklch(from var(--accent-blue) l c h / 0.2);
}
```

### 4. GLSL Shader Background (Cutting-Edge)
Untuk hero background yang benar-benar unik. Menggunakan WebGL.
```jsx
// Contoh dengan Three.js + React Three Fiber
// Ini yang dipakai maximeheckel.com
import { Canvas, useFrame } from '@react-three/fiber'
import { shaderMaterial } from '@react-three/drei'

const NoiseMaterial = shaderMaterial(
  { uTime: 0, uColor1: new THREE.Color('#4F8EF7'), uColor2: new THREE.Color('#A78BFA') },
  // vertex shader
  `void main() { gl_Position = vec4(position, 1.0); }`,
  // fragment shader — Simplex noise
  `uniform float uTime;
   uniform vec3 uColor1, uColor2;
   // ... noise function
   void main() {
     float n = snoise(vec3(vUv * 3.0, uTime * 0.3));
     gl_FragColor = vec4(mix(uColor1, uColor2, n * 0.5 + 0.5), 1.0);
   }`
)
```

### 5. Variable Font Animation
```css
/* Font berubah weight/width saat hover — sangat 2025 */
.kinetic-text {
  font-variation-settings: 'wght' 300, 'wdth' 75;
  transition: font-variation-settings 0.4s ease;
}
.kinetic-text:hover {
  font-variation-settings: 'wght' 900, 'wdth' 125;

/* Text yang "bernapas" */
@keyframes breathe {
  0%, 100% { font-variation-settings: 'wght' 300; }
  50%       { font-variation-settings: 'wght' 800; }

.breathing-text {
  animation: breathe 3s ease-in-out infinite;
}
```

### 7. Aurora / Mesh Gradient (Masih Kuat di 2025)
```css
.aurora-bg {
  background:
    radial-gradient(ellipse 80% 60% at 20% 40%, oklch(65% 0.2 250 / 0.15) 0%, transparent 60%),
    radial-gradient(ellipse 60% 80% at 80% 20%, oklch(65% 0.18 300 / 0.12) 0%, transparent 60%),
    radial-gradient(ellipse 70% 50% at 50% 90%, oklch(72% 0.17 160 / 0.10) 0%, transparent 60%),
    oklch(12% 0.03 250); /* dark navy */
  animation: aurora-shift 8s ease-in-out infinite alternate;
}

@keyframes aurora-shift {
  0%   { filter: hue-rotate(0deg); }
  100% { filter: hue-rotate(30deg); }
}
```

### 8. Glassmorphism Evolved (Lebih Subtle dari 2022)
```css
/* 2022: terlalu frosted, terlalu obvious */
/* 2025: subtle, refined */
.glass-card-2025 {
  background: oklch(100% 0 0 / 0.025);  /* hampir transparan */
  backdrop-filter: blur(40px) saturate(150%);
  border: 1px solid oklch(100% 0 0 / 0.06);
  box-shadow:
    0 0 0 1px oklch(100% 0 0 / 0.04),
    0 8px 32px oklch(0% 0 0 / 0.3),
    inset 0 1px 0 oklch(100% 0 0 / 0.08);  /* inner highlight */
}
```

## Inspirasi Narasi Konten

### Cara Menulis "About Me" yang Kuat

**Jangan begini (generic):**
> "I am a passionate fullstack developer with 10+ years of experience..."

**Tapi begini (storytelling):**
> "I started designing interfaces before I could write a line of code.
> For 4 years, I obsessed over pixel-perfect UIs — the spacing, the hierarchy, the way a button feels.
> Then I got curious about what happened when I hit 'submit'.
> That curiosity turned into a decade of writing the code behind the designs.
> Today I build products from the first sketch to the last deployment.
> That's not a common journey — and it's changed how I think about software."

### Formula Hero Tagline
```
[Unique descriptor] + [What you do] + [For whom / Impact]

Contoh:
"Designer-turned-engineer building products that feel as good as they work."
"I design systems, then build them — from figma to production."
"Full-stack developer with a designer's eye and an engineer's precision."
```
